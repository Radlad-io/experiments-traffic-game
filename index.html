Traffic Game


This project is based on a tutorial and the original version can be played here. In order to solidify what I learned in the tutorial I expanded the project to include music and sound effects. Remodeled the car to be a convertible. Used the sessionStorage API to store the users highscore. Added lighting and created a foggy night cycle which kicks in after the user reaches 10 laps.
